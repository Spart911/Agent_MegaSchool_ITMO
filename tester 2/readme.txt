Эта программа отправляет изображения на сервер с использованием HTTP POST запросов. Она поддерживает многопоточность для повышения скорости отправки.

Как использовать
Напиши go build main.go он ебанет тебе файл под твою систему.

Запустите программу: Дважды щелкни по файлу, чтобы открыть приложение.

Введите количество потоков: Когда программа запросит, введите количество потоков (например, 100) и нажмите Enter.

Результаты: После завершения отправки программа создаст файл отчета с именем report_DDMMYYYYHHMMSS.txt, в котором будут записаны ответы сервера.

Лицензия
Используйте и распространяйте программу на свой страх и риск.
